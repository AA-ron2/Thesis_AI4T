# Snellius Formula-AS Pipeline

This folder contains the Slurm-efficient execution path for `final_notebooks`.
The notebooks are now reporting notebooks: they read artifacts generated by
these scripts instead of running long training/evaluation loops themselves.

## MobaXterm / Snellius Usage

```bash
cd /home/<user>/thesis
git pull

module load 2023
module load Miniconda3/23.5.2-0
source "$(conda info --base)/etc/profile.d/conda.sh"
conda activate mysimenv

export PROJECT_DIR=/home/<user>/thesis
export DATA_DIR=/scratch-shared/<user>/datasets
export CONDA_ENV=mysimenv

bash scripts/snellius/submit_final_pipeline.sh
```

## Smoke Test

Runs tiny training jobs and only evaluates two held-out days. The final
comparison is skipped in smoke mode because the outputs are intentionally
partial.

```bash
SMOKE=1 bash scripts/snellius/submit_final_pipeline.sh
```

## Optional Flags

```bash
MAX_CONCURRENT=6                 # test-day array concurrency
B2_ALPHAS="0.1 1.0 10.0"         # B2 alpha jobs
TOTAL_TIMESTEPS=1000000          # override PPO timesteps
EVALUATION_ROLLOUTS=20           # Monte Carlo fill rollouts per test day
CVAR_N_WINDOWS=50                # B3 rollout-window CVaR samples
RUN_NB6=1                        # run feature ablations
RUN_SHAP=0                       # optional SHAP diagnostics
```
